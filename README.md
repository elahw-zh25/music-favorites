# 歌单歌曲收藏数查询
查询任意一个qq音乐歌单内的歌曲的收藏数量。    

## Prerequisite

- [ ] python  
- [ ] requests 可以使用pip进行安装。
```
pip install requests
```

## 使用方法   
参见`demo.py`。可直接在 MusicClient.py 相同路径下直接运行python，生成Client实例，调用`get_song_fav_playlist`并替换其中的歌单id和目标文件名。
得到的结果如`华语热门.tsv`所示。
